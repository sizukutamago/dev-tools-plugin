## User Stories

### Epic-001: ユーザー認証

#### US-001: ログインする

**Priority:** MVP
**Traceability:** Q-001

**As a** P-001 (一般ユーザー)
**I want to** メールアドレスとパスワードでログインする
**So that** サービスの機能を利用できる

##### Acceptance Criteria

- [ ] **AC-001-1** (正常系):
  ```gherkin
  Given 登録済みのメールアドレス「user@example.com」とパスワード「SecurePass123!」
  When ログインフォームに入力してログインボタンをクリック
  Then ダッシュボード画面に遷移する
  ```

- [ ] **AC-001-2** (異常系):
  ```gherkin
  Given 誤ったパスワード「WrongPass」
  When ログインボタンをクリック
  Then エラーメッセージが表示される
  ```

---

#### US-002: ログアウトする

**Priority:** MVP
**Traceability:** Q-002

**As a** P-001 (一般ユーザー)
**I want to** ログアウトする
**So that** セッションを終了できる

##### Acceptance Criteria

- [ ] **AC-002-1** (正常系):
  ```gherkin
  Given ログイン済みのユーザー
  When ログアウトボタンをクリック
  Then ログイン画面に戻る
  ```

- [ ] **AC-002-2** (異常系):
  ```gherkin
  Given ログイン済みのユーザー
  When セッションが期限切れ
  Then 自動的にログアウトされる
  ```
